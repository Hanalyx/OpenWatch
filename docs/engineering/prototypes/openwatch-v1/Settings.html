Maintenance

Pause scanning, connectivity checks, and alerting — fleet-wide, per group, or per host. A host is in maintenance if global, any of its groups, or the host itself is paused.

Global maintenance mode ?
Pause all scanning and alerting across every host. Use during planned infrastructure-wide work. Nothing is monitored while this is on.
Auto-resume
Automatically lift global maintenance after a set duration, so it can't be left on by accident.
Precedence is OR, not override. Putting a group into maintenance pauses every host in it, even if the host's own toggle is off. Each host shows why it's paused (global, a group name, or direct).
